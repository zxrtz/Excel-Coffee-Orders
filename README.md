# Excel-Coffee-Orders
Used Excel to clean and populate a Coffee Orders Relational Database

Population of Relational Databases Using:
LOOKUP, INDEX, MATCH, nested IF functions.

Graphics made using Pivot Tables, and Pivot Charts.
